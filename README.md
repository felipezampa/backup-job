# Backup Pentaho

<p>O objetivo deste código é realizar backup de forma automatizada de arquivos pentaho</p>
<p>O programa salva um arquivo ZIP com nome do projeto e data no formato: <strong>Job_Projeto_22.11.2022</strong></p>

Versão: 1.0
